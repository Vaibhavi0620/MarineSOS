# 🚢 MarineSOS

MarineSOS is a React.js and Firebase based maritime emergency response web application that allows users to quickly send SOS requests with their live location.

## Features

- 🔐 User Authentication (Firebase Authentication)
- 👤 User Profile Management
- 🚨 Send SOS with Live GPS Location
- 📍 Geolocation Integration
- 📋 SOS History
- 🔒 Protected Dashboard
- 📱 Responsive User Interface

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- Firebase Authentication
- Cloud Firestore
- React Router DOM
- Geolocation API

## Installation

```bash
git clone https://github.com/yourusername/MarineSOS.git

cd MarineSOS

npm install

npm run dev
```
